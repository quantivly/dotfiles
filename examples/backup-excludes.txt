# Syntax: one pattern per line; a leading "/" anchors to the snapshot root;
# "**" matches across directories. Full-line "#" comments and blank lines are
# ignored. IMPORTANT: restic does NOT strip INLINE comments — a "# ..." after a
# pattern becomes part of the pattern and silently disables the exclude. Keep
# every comment on its own line.

# `exclude-caches = true` (set in profiles.toml) additionally skips any dir
# containing a CACHEDIR.TAG. Rule of thumb: exclude ONLY cheaply regenerable
# data. Never add ~/.ssh, ~/.gnupg, ~/.config, ~/.local/share/keyrings,
# ~/.dotfiles, or ~/.config/AWSVPNClient.
